## A3 â€” C Code Assessment

All C source files (`*.c`) and header files (`*.h`) have been moved to `legacy_c/`.

**Rationale**: The `pynetphorest` Python package (a modern Python wrapper) is now the
primary motif scorer (`motif_scoring.py` uses `pynetphorest.core`).  The compiled
`netphorest` binaries are retained in `binary_human_public/`, `binary_universal_public/`,
and `netphorest/` for backward compatibility with the `--netphorest` CLI flag but are
no longer the primary execution path.

---

## A4 â€” CGI / Perl Assessment

All Perl scripts (`.pl`) have been moved to `legacy_web/`.

**Rationale**: These scripts (`insr2h.pl`, `nn2h.pl`, `pssm2h.pl`) are code-generators
that produce C header files from binary model data.  They are part of the legacy C build
system and are not needed when using `pynetphorest`.
